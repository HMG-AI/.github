# HMG-AI

HMG builds governed, durable memory infrastructure for AI agents: capture useful context, retrieve it with scope and provenance, and correct or govern it over time.

## Public projects

- [HMG-public](https://github.com/HMG-AI/HMG-public) — protocol, SDKs, documentation, and releases.
- [HMG-Benchmark](https://github.com/HMG-AI/HMG-Benchmark) — reproducible memory benchmarks and evaluation assets.
- [HMG website frontend](https://github.com/HMG-AI/HMG-website-frontend) — the public product website.

## Contributing and security

Start with the repository's issue forms and contribution guide. Please report vulnerabilities through GitHub private vulnerability reporting when it is available, or follow our [security policy](https://github.com/HMG-AI/.github/blob/main/SECURITY.md). Do not open a public issue for a suspected vulnerability.
